# SESI 2: TEKNIK PROMPT DASAR
## Cara Memberikan Instruksi yang Tepat
**Durasi: 45 menit**

---

## Apa itu Prompt?

**Prompt = Instruksi yang Anda berikan ke ChatGPT**

Seperti memesan makanan:
- ❌ "Makan" → tidak jelas
- ✅ "Nasi goreng pedas, tanpa telur" → jelas & detail

**Prompt yang baik = Hasil yang baik!**

---

## Formula 4W: Prompt yang Efektif

**WHO** (Siapa) → Peran ChatGPT  
**WHAT** (Apa) → Tugas yang diminta  
**WHERE** (Konteks) → Situasi/latar belakang  
**WHICH** (Format) → Bentuk output yang diinginkan

---

## Contoh Prompt BURUK ❌
```
Buatkan surat
```

**Kenapa buruk?**
- Surat apa?
- Untuk siapa?
- Keperluan apa?

**Hasilnya:** ChatGPT bingung, output tidak sesuai

---

## Contoh Prompt BAIK ✅
```
Anda adalah staff TU Fakultas Ilmu Komputer. 
Buatkan surat undangan rapat evaluasi semester 
untuk seluruh dosen. Rapat pada Jumat, 20 Desember 2024, 
pukul 09.00 WITA di Ruang Senat. 
Format surat formal dengan kop surat.
```

**Hasilnya:** Surat lengkap dan sesuai!

---

## Bedah Formula 4W

**WHO:** "Anda adalah staff TU Fakultas Ilmu Komputer"  
→ ChatGPT tahu posisinya

**WHAT:** "Buatkan surat undangan rapat evaluasi semester"  
→ Tugas jelas

**WHERE:** "untuk seluruh dosen. Rapat Jumat, 20 Des, 09.00, Ruang Senat"  
→ Konteks lengkap

**WHICH:** "Format surat formal dengan kop surat"  
→ Bentuk output jelas

## Template Prompt untuk Surat
```
Anda adalah [jabatan Anda] di [unit kerja].
Buatkan [jenis dokumen] tentang [topik].
Tujuan: [untuk siapa/apa].
Detail: [tanggal/waktu/tempat/hal penting lainnya].
Format: [formal/resmi/tabel/dll].
```

**Copy template ini!**

---

## DEMO LANGSUNG: Surat Undangan

**Kasus:** Undangan rapat koordinasi bagian akademik

**Prompt:**
```
Anda adalah staff akademik Fakultas Ilmu Komputer UMI.
Buatkan surat undangan rapat koordinasi persiapan ujian akhir semester
untuk Ketua Program Studi S1 dan D3.
Rapat: Senin, 25 November 2024, pukul 10.00 WITA di Ruang Dekan.
Format surat formal dengan nomor surat.
```

**Mari kita lihat hasilnya!**

---

## Tips Memperbaiki Output

Jika hasil kurang sesuai:

**Cara 1:** Ketik lanjutan
```
Tambahkan agenda rapat
```

**Cara 2:** Minta revisi
```
Ubah bahasanya lebih formal
```

**Cara 3:** Lebih spesifik
```
Sertakan nama penandatangan: Dr. Ahmad, M.Kom
```

## Contoh untuk Dokumen Lain

**Surat Tugas:**
```

Budi Santoso mengikuti pelatihan di Jakarta 
tanggal 5-7 Desember 2024. Format resmi dengan nomor surat.
```

**SK Panitia:**
```
Anda adalah staff TU. Buatkan SK Panitia Seminar Nasional
dengan ketua: Dr. Ani, sekretaris: Budi, M.Kom, 
bendahara: Citra, S.Kom. Format SK formal.
```

## PRAKTEK: Buat Surat Anda! ⏱️

**25 Menit**

**Tugas:** Buat 1 surat sesuai bidang Anda:
- **TU:** Surat undangan/surat tugas
- **Akademik:** Surat pemberitahuan jadwal
- **Keuangan:** Surat permohonan pencairan
- **Perlengkapan:** Surat permohonan barang

**Gunakan Formula 4W!**

## Cek Hasil Anda

✅ Apakah formatnya sesuai?  
✅ Apakah informasinya lengkap?  
✅ Apakah bahasanya formal?  
✅ Apakah ada yang perlu ditambah?
