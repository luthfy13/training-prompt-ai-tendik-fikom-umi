## Teknik Revisi: Instruksi Lanjutan

**Jangan mulai chat baru!** Cukup ketik lanjutan di chat yang sama.

**Contoh Revisi:**

**Output pertama:** Surat terlalu panjang  
**Instruksi lanjutan:** 
```
Persingkat menjadi 1 halaman saja
```

**Output kedua:** Bahasa terlalu formal  
**Instruksi lanjutan:**
```
Sederhanakan bahasanya agar lebih mudah dipahami
```

---

## Contoh Revisi: Tambah Detail

**Prompt awal:**
```
Buatkan surat undangan rapat untuk dosen
```

**Output:** Surat dasar, kurang detail

**Revisi 1:**
```
Tambahkan agenda rapat:
1. Pembahasan jadwal UAS
2. Evaluasi semester
3. Rencana akreditasi
```

**Revisi 2:**
```
Sertakan dresscode: kemeja/batik
```

---

## Contoh Revisi: Ubah Format

**Output:** Paragraf panjang susah dibaca

**Instruksi revisi:**
```
Ubah dalam bentuk poin-poin
```

**Output:** Tabel angka tidak rapi

**Instruksi revisi:**
```
Rapikan tabelnya dengan border yang jelas
```

**Output:** Terlalu formal

**Instruksi revisi:**
```
Gunakan bahasa yang lebih akrab tapi tetap sopan
```

# BAGIAN 2: TIPS & TRIK LANJUTAN

---

## Tip 1: Minta Beberapa Alternatif

Tidak puas dengan output pertama?

**Ketik:**
```
Buatkan 3 versi alternatif
```

**Hasilnya:** ChatGPT beri 3 pilihan berbeda!  
Anda tinggal pilih yang paling cocok.

---

## Tip 2: Minta Penjelasan

Tidak paham istilah dalam output?

**Ketik:**
```
Jelaskan apa maksud "dengan ini" dan 
"sehubungan dengan hal tersebut"
```

ChatGPT akan jelaskan + kasih contoh penggunaan.

---

## Tip 3: Gabungkan Dokumen

Punya 2 draft yang bagus? Gabungkan!

**Contoh:**
```
Gabungkan draft surat undangan yang pertama dan kedua.
Ambil pembukaan dari draft 1, isi dari draft 2, 
dan penutup dari draft 1.
```

---

## Tip 4: Terjemahkan Bahasa

Dapat surat/dokumen bahasa Inggris?

**Ketik:**
```
Terjemahkan dokumen ini ke Bahasa Indonesia formal
```

**Atau sebaliknya:**
```
Terjemahkan surat ini ke Bahasa Inggris
untuk keperluan korespondensi internasional
```

---

## Tip 5: Cek Ejaan & Tata Bahasa

Sudah buat surat sendiri tapi takut ada typo?

**Ketik:**
```
Cek ejaan dan tata bahasa dokumen ini, 
perbaiki jika ada kesalahan:
[paste dokumen Anda]
```

ChatGPT = proofreader gratis!

---

## Tip 6: Ringkas Dokumen Panjang

Dapat dokumen 10 halaman, perlu ringkasan?

**Ketik:**
```
Ringkas dokumen ini menjadi poin-poin penting,
maksimal 1 halaman
[paste dokumen]
```

Hemat waktu membaca!

---

## Tip 7: Buat Email dari Surat Resmi

Sudah punya surat formal, perlu kirim via email?

**Ketik:**
```
Ubah surat resmi ini menjadi format email 
yang lebih ringkas dan informal
```

---

## Tip 8: Konsistensi Nomor Surat

**Trik untuk nomor surat:**
```
Gunakan format nomor: 
123/UN.56.2/TU/2024
Keterangan:
- 123 = nomor urut
- UN.56.2 = kode fakultas
- TU = bagian
- 2024 = tahun
```

ChatGPT akan ikuti format ini!

# BAGIAN 4: TEMPLATE SIAP PAKAI

## Template 1: Surat Resmi Umum

```
- Tujuan: [untuk siapa/keperluan apa]

- Tempat: [lokasi]
- [info penting lainnya]
```

**Copy template ini ke notes HP/komputer Anda!**

---

## Template 2: Proposal/Permohonan
